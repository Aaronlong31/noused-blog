在Ubuntu中使用goagent

  • 安装 python: sudo apt-get install python
  • 安装GAE for linux，下载地址: https://developers.google.com/appengine/downloads , 下载后解压到/home/zhanglong/tools/google_appengine
  • goagent现在托管在github上，地址: https://github.com/goagent/goagent ， 最好fork一下，创建自己的一份copy，以后修改配置文件就修改自己的好了。以我的为例，我的goagent地址: https://github.com/Aaronlong31/goagent
  • 下载goagent到google_appengine目录: cd /home/zhanglong/tools/google_appengine/, git clone git@github.com:Aaronlong31/goagent.git
  • 服务器端上传 cd /home/zhanglong/google_appengine, sudo python appcfg.py update goagent/server/python
  • 客户端使用，修改 /home/zhanglong/tools/google_appengine/goagent/local/proxy.ini中的appid，然后在local目录中运行sudo python proxy.py。
  • 在Ubuntu Dash中添加快捷方式，在/usr/share/applications/中创建goagent.desktop文件，添加以下内容：

<html><body>You are being <a href="https://gist.github.com/Aaronlong31/4705697/raw/">redirected</a>.</body></html>
然后就可以在dash中看到goagent了。
