Use Highlight in Github Page

Install pygment

sudo apt-get install python-setuptools
sudo easy_install Pygments

Create style file

you can generate it by Pygments:

pygmentize -S default -f html > css/pygments/highlight.css

default is a style. Pygments ships some builtin styles which are maintained by the Pygments team. To get a list of known styles you can use this snippet:

>>> from pygments.styles import STYLE_MAP
>>> STYLE_MAP.keys()
['default', 'emacs', 'friendly', 'colorful']
['monokai', 'manni', 'rrt', 'perldoc', 'borland', 'colorful', 'default', 'murphy', 'vs', 'trac', 'tango', 'fruity', 'autumn', 'bw', 'emacs', 'vim', 'pastie', 'friendly', 'native']

Add the css file to _layout/default.html.

<link rel="stylesheet" href="/css/pygments/highlight.css" />

Modify your _config.yml add this:

pygments: true

Write code

To denote a code block that should be highlighted:

``` textile

your code here

```

Done!
